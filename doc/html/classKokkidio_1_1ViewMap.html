template<typename _EigenType , Target targetArg = DefaultTarget>
KOKKOS_FUNCTION auto Kokkidio::ViewMap< _EigenType, targetArg >::map ( ) const -> MapType
inline

Returns an Eigen::Map to memory on the ViewMap's target (Target::host or Target::device).

This represents the core functionality of an ViewMap, because this Eigen::Map can be used in any Eigen operation.

If the ViewMap was initialised with an Eigen object obj and target == Target::host, then the Eigen::Map uses the same data as that obj. Otherwise, it uses data of a managed Kokkos::View.

template<typename _EigenType , Target targetArg = DefaultTarget>
KOKKOS_FUNCTION auto Kokkidio::ViewMap< _EigenType, targetArg >::view ( ) const -> ViewType
inline

Returns the stored Kokkos::View. If the ViewMap was initialised with an Eigen object obj, this returns an unmanaged View with the same data pointer as obj. Otherwise, it returns a managed view, whose memory space is Kokkos::DefaultExecutionSpace, if target == Target::device, and Kokkos::DefaultHostExecutionSpace, if target == Target::host.
